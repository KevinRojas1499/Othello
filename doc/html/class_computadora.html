Detailed Description

Colección de estrategias para la computadora.

Cada instancia se debe construir especificando sobre cuál tablero debe jugar, qué estrategia debe usar y cuál color usa.

Para que la computadora realice una jugada, se debe invocar al método jugada().

Actualmente existen dos estrategias:

  • La estrategia número 0 pone la ficha en el primer lugar que encuentra, buscando de derecha a izquierda y de arriba a abajo.
  • La estrategia no. 1 busca la posición donde pueda comer la mayor cantidad de fichas posible.
